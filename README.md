# React E-commerce Shop

A modern, React-only storefront that pairs a curated capsule collection with an expressive UI, instant cart updates, and a checkout form that feels premium.

## Features
- Responsive hero, highlights, and product gallery styled with gradients, glassmorphism, and intentional whitespace.
- Interactive cart preview that tracks subtotal, shipping, tax, and grand total with quantity controls.
- Checkout form that captures name, email, and shipping address, then confirms the order with a friendly message.
- Easy commands for development, production builds, and previews.

## Getting started
1. `npm install`
2. `npm run dev`
3. Visit `http://localhost:5173` to explore the experience.
4. `npm run build` to create an optimized production bundle.

## Stack
- React 18 + Vite 5
- Vanilla CSS for the UI system
